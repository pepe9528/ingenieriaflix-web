Curso de JavaScript

Aprende a crear aplicaciones web interactivas con JavaScript.

JavaScript es el lenguaje de programación más usado en el mundo y es esencial para el desarrollo web. Con JavaScript puedes hacer que tus páginas web sean interactivas, crear aplicaciones completas, trabajar con APIs y mucho más. Esta página reúne los mejores videos en español para aprender JavaScript desde cero hasta un nivel intermedio-avanzado.

JavaScript desde Cero — Curso Completo

El mejor punto de partida para aprender JavaScript. Este curso completo cubre todos los fundamentos: variables, tipos de datos, operadores, condicionales, bucles y funciones explicados de forma clara y práctica.

Variables y Tipos de Datos

Aprende cómo declarar variables con var, let y const, cuáles son los tipos de datos en JavaScript (strings, números, booleanos, arrays y objetos) y cuándo usar cada uno.

Condicionales y Bucles

Las estructuras de control son la base de cualquier programa. Aprende if, else, switch, for, while y do-while con ejemplos prácticos para tomar decisiones y repetir acciones en tu código.

Funciones en JavaScript

Las funciones son bloques de código reutilizables y son fundamentales en JavaScript. Aprende funciones declaradas, expresadas, arrow functions y el concepto de scope.

Arrays y Objetos

Los arrays y objetos son las estructuras de datos más importantes de JavaScript. Aprende a crearlos, modificarlos y recorrerlos con los métodos más usados como map, filter y reduce.

Manipulación del DOM

El DOM es la interfaz que permite a JavaScript interactuar con el HTML de la página. Aprende a seleccionar elementos, modificar su contenido, cambiar estilos y crear elementos dinámicamente.

Eventos en JavaScript

Los eventos hacen que tu página reaccione a las acciones del usuario: clics, teclas, movimientos del mouse y más. Aprende addEventListener y cómo manejar eventos de forma eficiente.

JavaScript Asíncrono — Promesas y Async/Await

El código asíncrono permite hacer peticiones a servidores sin bloquear la página. Aprende callbacks, promesas y async/await, conceptos clave para trabajar con APIs y datos externos.

Proyecto — Calculadora con JavaScript

Aprende haciendo. En este proyecto crearás una calculadora funcional usando HTML, CSS y JavaScript. Un ejercicio clásico e ideal para consolidar todo lo que aprendiste sobre el DOM y eventos.

Proyecto Completo — Aplicación con JavaScript

Pon en práctica todo lo aprendido creando una aplicación completa con JavaScript. Manejo del DOM, eventos, arrays y lógica de programación aplicada a un proyecto real.
